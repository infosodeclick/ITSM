# Roadmap

This project should grow as a modular ITSM inventory system. Each phase must keep the app deployable and must pass lint/build before release.

## Phase 1: Foundation

- Users, roles, permissions
- Branches, departments, employees
- Asset categories and vendors
- Expanded asset profile fields
- Audit log data model
- Sidebar navigation shell

## Phase 2: Core Workflows

- Automatic asset code generation
- Assign asset to employee
- Transfer asset
- Return asset
- Asset detail page with history tabs
- HR onboarding request
- HR offboarding request

## Phase 3: Alerts and Reports

- Warranty alerts
- License alerts
- Excel export
- Report filters
- In-app notifications
- Audit log viewer

## Phase 4: Advanced Operations

- Approval flow
- QR code and barcode
- Digital acceptance
- PDF reports
- Excel import
- Budget and cost dashboard
- LINE, Teams, or Telegram notification channels
